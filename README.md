# scala-cli-maven-surefire-spring-findbugs-mockito-cucumber-testng-car-data

## Description
A POC for maven app using testng
mockito, and cucumber framework
 with findbugs
and urefire plugin.

## Tech stack
- scala
- maven
	- spring
  - testng
  - findbugs
  - surefire
  - cucumber
	- mockito


## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
